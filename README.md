# 📊 数研工坊 — A股多维趋势分析 & 自定义策略引擎平台

> 不堆砌"AI量化"噱头，先打牢投研地基，再生长出真正可用的量化交易系统。

## 🏗 设计理念：先筑地基，再建大厦

市面上多数产品一上来就宣称"AI量化自动交易"，往往跳过了最关键的**市场认知与趋势研判环节**。  
本项目的核心理念是**稳扎稳打分层构建**：

- **第一层 · 宏观与板块趋势**  
  跟踪大盘运行态势、行业景气度轮动、热点概念强弱，建立大周期方向感。

- **第二层 · 自研趋势指标体系**  
  在经典均线/MACD/RSI之外，内置自研的**实时强弱得分、趋势得分、动态均线偏离度**等维度，量化"当下市场是在走强还是走弱"。

- **第三层 · 策略引擎（量化交易系统）**  
  在上述地基之上，推出**策略引擎（Strategy Engine）**——你可基于已有趋势指标组合条件，设定买入上限比例、止盈/止损幅度，由引擎自动记录策略交易明细、生成历史趋势图与胜率统计。  
  > 策略引擎创建后不可修改，保障历史回测与跟踪记录的公正透明。

我们坚信：**只有理解了趋势从何而来，量化策略才不是黑箱赌博。**
